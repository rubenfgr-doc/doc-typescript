# TypeScript - ECMAScript 20XX

```sh
# Instalar dependencias
npm install

# Construir el paquete web
npx webpack 
# or
node ./node_modules/webpack/.bin/webpack.js

# iniciar el proyecto
npm start
```

## Cambiar el puerto

./package.json

```json
"start": "webpack-dev-server --mode development --open --port=8081"
```
